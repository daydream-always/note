@FeignClient(value = "service-customer")

@GetMapping("/customer/info/login/{code}")

public interface CustomerInfoFeignClient

public Result<Long> login(@PathVariable String code);























@FeignClient

@FeignClient 是 Spring Cloud OpenFeign 提供的一个注解，它的主要作用是：

声明 Feign 客户端、服务发现与负载均衡、简化 HTTP 调用、集成 Spring MVC 注解

​	将 CustomerInfoFeignClient 接口标记为一个 Feign 客户端接口，用于调用名为 service-

customer 的微服务

​	通过 value = "service-customer" 指定要调用的服务名称，结合服务注册中心（如 Nacos、

Eureka）实现服务发现，自动支持负载均衡，可以在多个服务实例间进行请求分发

​	配合接口中的 @GetMapping、@PostMapping 等注解，自动生成 HTTP 请求客户端实现

开发者只需关注业务逻辑，无需手动编写 HTTP 请求代码

​	可以直接使用 @PathVariable、@RequestBody 等 Spring MVC 注解，实现 RESTful API 的

声明式调用

这样，当调用 CustomerInfoFeignClient.login() 方法时，会自动向 service-customer 服务

发起 HTTP GET 请求访问 /customer/info/login/{code} 接口。
