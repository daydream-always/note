管理员后端部署

安装jdk

* 需要在`server01`中安装JDK，本项目采用JDK17
* 解压JDK安装包，将jdk解压到`/opt`目录
* 测试JDK安装效果`/opt/jdk-17.0.8/bin/java -version`

部署

* jar包上传到`server01`服务器的`/opt/lease`目录下，若目录不存在，自行创建即可
* 集成Systemd，方便项目的启动、停止或者重启，我们同样使用Systemd来管理后端服务的进程
* 后台管理系统集成Systemd，创建`lease-admin.service`文件，`vim /etc/systemd/system/lease-admin.service`
* 写入内容
* 启动项目`systemctl start lease-admin`





管理员前端部署

打包

* npm run build，生成一个dist文件夹，包含所有静态资源

部署

* 上传dist文件夹至`server02`服务器的`/usr/share/nginx/html/admin`
* 创建`/etc/nginx/conf.d/admin.conf`文件
* 重新加载Nginx配置文件`systemctl reload nginx`
* 访问http://admin.lease.atguigu.com
