如何在centos7系统安装docker软件？

一、安装前准备（必做步骤）

1. 卸载旧版 Docker（若存在）

2. 安装依赖工具

3. 配置 Docker 官方 yum 源

二、安装 Docker（核心步骤）
1. 安装 Docker CE（社区版，免费常用）

2. 启动 Docker 服务并设置开机自启

三、验证 Docker 安装成功
1. 查看 Docker 版本
2. 运行测试容器（验证功能正常）

Docker 默认境外镜像源国内访问超时，导致无法拉取 hello-world 镜像。Docker 默认使用的官方镜像仓库在境外，国内访问速度慢甚至无法连接。解决方法是将 Docker 的镜像源替换为国内的镜像加速器（如阿里云、网易、腾讯云等）。

```
{
    "registry-mirrors": [
        "https://docker.m.daocloud.io",
        "https://1ms.run",
        "https://dockerproxy.com",
        "https://docker.mirrors.ustc.edu.cn",
        "https://docker.nju.edu.cn",
        "https://iju9kaj2.mirror.aliyuncs.com",
        "http://hub-mirror.c.163.com",
        "https://cr.console.aliyun.com",
        "https://hub.docker.com",
        "http://mirrors.ustc.edu.cn"
    ]
}
```
