期末项目考核

1.主题与核心知识点

项目主题： 基于ResNet 深度特征提取和模糊聚类增强的人脸识别系统

核心知识点：

深度学习特征提取： ResNet (残差网络)

模式识别算法： 模糊 C 均值聚类 (FCM)

应用场景： 人脸识别 (Face Recognition)

推荐数据集： ORL (Olivetti Research Laboratory) 人脸数据库或类似规模的人脸数据集。

2. 项目交付物要求

学生需提交以下两类核心交付物：

源代码文件：完整的 Python 代码，包含 ResNet 特征提取、FCM 算法实现或调用、分类评估等。代码需可运行、注释完整、结构清晰。

项目报告：规范的 PDF 文档（建议 5000-8000字），需包含理论基础、方法论、对比实验设计、结果分析与讨论。

3. 期末项目评分细则

期末项目的分数将严格按照 代码实现（50%） 和 报告分析（50%） 的比例进行细分评估。

关键考核点

I. 💻 代码实现与运行 (25%)

1. ResNet 特征提取正确性10%。**ResNet 模型调用、特征向量提取（Embedding）**的正确性。


2. FCM 算法实现10%。模糊 C 均值算法的正确实现及其**在特征空间中的应用**。


3. 代码规范性5%。代码可读性、注释、命名规范、运行效率。

II. 📄 报告结构与理论 (25%)

1. 理论原理阐述深度10%。ResNet 和 FCM 核心原理（如**目标函数、隶属度**）的准确描述。


2. 报告完整性与规范性5%。报告结构完整、逻辑清晰、语言表达专业。


3. 方法论描述清晰度10%。清晰描述 **ResNet 提取层级**和 **FCM 算法的实施细节**。

III. 📊 实验对比与分析 (50%)

1. 对比实验设计15%。必须包含**基准模型 与 FCM 增强模型 的性能对比**。


2. 定量结果分析15%。**准确率、F1-Score 等指标的表格对比**，并对 FCM 的性能提升进行论证。


3. 可视化分析15%。**使用 t-SNE/PCA 对特征空间进行可视化**；**展示并分析 FCM 隶属度分布图**，识别模糊样本。


4. 讨论与展望5%。对项目成果的总结、限制条件的讨论以及未来改进方向的建议。
