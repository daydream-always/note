

4. 以下哪种集合类可以用于实现先进先出（FIFO）操作？ ( )
A. HashSet
B. ArrayList
C. LinkedList
D. TreeSet



正确答案是：**C. LinkedList**

------

### 解析：

先进先出（**FIFO, First In First Out**）是一种队列行为，即最早加入的元素最先被取出。

我们逐个分析选项：

- **A. HashSet** ——无序集合、元素乱序
  - 无序集合，不保证元素的插入或访问顺序，**不能用于 FIFO**。
  - 
- **B. ArrayList**  ——数组，支持索引访问，但不适合插入删除操作
  - 虽然底层是数组、支持按索引访问，但若用作队列（在头部删除、尾部添加），效率很低（因为删除头部元素需要移动大量元素），**一般不用于实现 FIFO**。
  - 
- **C. LinkedList**  ——实现deque接口，既可以作为双向链表，也可以作为队列
  - 实现了 `Deque` 接口，既可以作为**双向链表**，也可以当作**队列（Queue）**使用。  
  - 提供如 `addLast()`（入队）、`removeFirst()`（出队）等方法，**天然支持高效的 FIFO 操作**。
    ✅ **适合实现 FIFO**。
  - 
- **D. TreeSet**  ——基于红黑树的有序集合，按元素自然顺序排序，与插入顺序无关
  - 是一个有序集合（基于红黑树），按元素的自然顺序或自定义比较器排序，**与插入顺序无关**，**不能用于 FIFO**。

------

### 补充说明：

Java 中若明确要使用 FIFO 队列，通常会这样写：

```java
Queue<Integer> queue = new LinkedList<>();
```

因此，**选项 C 是最合适的答案**。
