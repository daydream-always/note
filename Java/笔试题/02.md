

2. 以下哪种集合类不能存储重复的元素？ ( )
   A. ArrayList
   B. HashSet
   C. LinkedList
   D. Vector



正确答案是：**B. HashSet**

### 解析：

- **ArrayList**：基于动态数组实现，**允许存储重复元素**，并且**保持插入顺序**。
- **LinkedList**：基于双向链表实现，也**允许重复元素**，同样**保持插入顺序**。
- **Vector**：与 ArrayList 类似，是线程安全的动态数组，**允许重复元素**。
- **HashSet**：基于哈希表实现的 Set 接口类，**不允许存储重复元素**（根据 `equals()` 和 `hashCode()` 判断），且**不保证顺序**。

> Java 中，**Set 接口的实现类（如 HashSet、LinkedHashSet、TreeSet）都不允许重复元素**；而 **List 接口的实现类（如 ArrayList、LinkedList、Vector）都允许重复元素**。

因此，本题选 **B. HashSet**。
