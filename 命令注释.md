#### 1. 如果你用的是 Windows CMD（你当前的环境）

CMD 的注释符号是 `rem`（remark 的缩写），想要给命令加注释，有两种写法：

bash

```bash
# 写法1：注释单独一行（推荐，清晰）
rem 查看远程仓库
git remote -v

# 写法2：用 & 连接命令和注释（同一行）
git remote -v & rem 查看远程仓库
```

#### 2. 如果你用的是 Git Bash / Linux/macOS 终端

这些环境支持 `#` 作为注释符，你原本的写法就生效：

bash

```bash
git remote -v # 查看远程仓库
```
