简要罗列Java数组的知识点，同时做简要讲解，并说明这些知识在实际开发有哪些应用场景

### 1. 数组基本概念

**定义**：数组是存储相同类型元素的固定大小的容器，元素在内存中连续存储。

**特点**：

- 长度固定
- 元素类型相同
- 通过索引访问元素（从0开始）
- 数组名指向数组首地址

### 2. 数组声明与初始化

// 声明
int[] arr1;
int arr2[];
String[] names;

// 初始化
int[] arr = new int[5];        // 创建长度为5的数组
int[] arr = {1, 2, 3, 4, 5};   // 直接赋值
int[] arr = new int[]{1, 2, 3}; // 传统方式

### 3. 数组遍历

// 普通循环（下标拿值）
for(int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}

// 增强for循环（直接变量拿值）
for(int element : arr) {
    System.out.println(element);
}

### 4. 数组常用方法

- `length`：获取数组长度
- `Arrays.sort()`：排序
- `Arrays.equals()`：比较数组
- `Arrays.fill()`：填充数组

### 5. 多维数组

int[][] matrix = new int[3][4];  // 3行4列 

int[][] matrix = {{1,2},{3,4},{5,6}};  // 直接初始化
