# 改用 SSH 方式推送 GitHub（更稳定，避免证书问题）

1. 按之前的方法生成 SSH 密钥，并添加到 GitHub 账号；
2. 将 GitHub 远程仓库地址改为 SSH 格式：

# 先删除原有的 GitHub HTTPS 地址
git remote rm github
# 添加 SSH 格式的 GitHub 地址
git remote add github git@github.com:daydream-always/note.git







# 拉取 GitHub 远程仓库的 main 分支到本地，并自动合并
git pull github main

git push github main



# 核心命令：允许合并无关联历史，拉取并合并 GitHub main 分支
git pull github main --allow-unrelated-histories

git push github main
